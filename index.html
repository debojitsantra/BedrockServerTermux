Bedrock Server on Android

Beginner-friendly guide to hosting a Minecraft Bedrock server using Termux and Ubuntu on Android.

Runs fully on your phone. No PC required.
Overview

What You Will Do

This guide shows you how to run a Minecraft Bedrock server on your Android device using Termux and an Ubuntu environment. You will:

  • Install Ubuntu inside Termux
  • Download and set up the Bedrock server
  • Use Playit to expose your server to the internet
  • Update the server when a new version is available
Good to know You do not need root access. Everything runs inside Termux and a proot Ubuntu environment.
Setup

Prerequisites

  • Android phone or tablet with enough free storage and RAM
  • Termux installed (F-Droid version is recommended)
  • Stable internet connection
  • Very basic command line usage (copy/paste commands is enough)
Tip In Termux, long-press to paste commands you copied. It makes life easier and avoids typing mistakes.
Step 1

Install Ubuntu in Termux

1. Open Termux

Open the Termux app on your Android device. Make sure you are connected to the internet.

2. Update Termux and install Ubuntu setup script

Run these commands one by one:

apt update -y
apt upgrade -y
apt install wget -y
wget https://raw.githubusercontent.com/debojitsantra/BedrockServerTermux/refs/heads/main/setup_ubuntu.sh
bash setup_ubuntu.sh

The script will download and configure an Ubuntu environment inside Termux using proot.

Note This step may take a while depending on your internet speed. Let it finish completely.

3. Log in to Ubuntu

After the installation is complete, log into Ubuntu with:

proot-distro login ubuntu

Your prompt should now look different, indicating you are inside the Ubuntu environment (for example, the path might start with /root).

Step 2

Download Environment and Bedrock Server

Make sure you are inside the Ubuntu session (you should have run proot-distro login ubuntu).

1. Update Ubuntu and download setup script

Run these commands:

apt update -y
apt upgrade -y
apt install wget -y
wget https://raw.githubusercontent.com/debojitsantra/BedrockServerTermux/refs/heads/main/setup_env.sh
bash setup_env.sh

The setup_env.sh script will:

  • Install required packages like Box64, Playit, and Git
  • Download the Minecraft Bedrock server files
  • Unzip and prepare the server folder
Tip If a command fails, check your internet connection and run apt update -y once more, then retry the script.
Step 3

Run the Minecraft Server

To keep both the server and Playit running, you will use two Termux sessions.

1. Open two Termux sessions

  • Open Termux as usual. This is Session 1.
  • Open the side menu or use the Termux plus button to create Session 2.

2. Log in to Ubuntu in both sessions

In each session, run:

proot-distro login ubuntu

3. Start the Minecraft server (Session 1)

In Session 1 (Ubuntu already logged in), run:

cd
./run

This should start the Bedrock server. The terminal will show log messages from the server. Keep this session open and running.

Important Make sure you are in the /root directory inside Ubuntu before running ./run. Using cd with no arguments usually takes you there.

4. Start Playit tunnel (Session 2)

In Session 2 (also inside Ubuntu), run:

playit

This will start the Playit client, which creates a tunnel so players on the internet can connect to your local server.

Step 4

Connect Your Server to the World

1. Get the Playit link

When you run playit, it will print a link in the terminal. That link is for configuring your tunnel.

2. Open the link in your browser

  • Copy the Playit link from Termux.
  • Open it in your browser on your phone or another device.
  • Sign in or create an account on playit.gg.
  • Follow the instructions to configure a Minecraft Bedrock tunnel.

After configuration, Playit will give you an address (host and port) that your friends can use in their Minecraft Bedrock client to connect to your server.

Tip Share the address from Playit with your friends. They can enter it as an external server in Minecraft Bedrock.
Update

Updating the Bedrock Server

When a new version of the Bedrock server is available, you can update automatically from inside Ubuntu.

1. Go to the server directory

From inside the Ubuntu session, run:

cd server
wget https://raw.githubusercontent.com/debojitsantra/BedrockServerTermux/refs/heads/main/update.sh
chmod +x update.sh
./update.sh

The script will download and install the latest version of the Bedrock server for you.

Note Make sure the server is stopped before updating to avoid file conflicts. You can stop the server by using the server console or closing the session where it is running.
Help

Tips and Troubleshooting

Common checks

  • Server not starting: Make sure you are in the correct directory (usually /root) before running ./run.
  • Script errors: Run apt update -y and apt upgrade -y in Ubuntu, then try the script again.
  • Playit not connecting: Check that the Minecraft server is running, and that you completed the tunnel setup on the Playit website.
Do not close Termux If you close Termux or kill the app from recent apps, your server and Playit tunnel will stop. Keep Termux running in the background while players are connected.

Quick sanity checklist

  • Termux installed and up to date
  • Ubuntu installed via setup_ubuntu.sh
  • Environment and server installed via setup_env.sh
  • Server started with ./run in Session 1
  • Playit running with playit in Session 2
  • Playit tunnel configured on the website
